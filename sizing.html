Units

Absolute Units

px
pixels, standard static unit. Good for instances where a set size is required. Also frequently used for spacing and smaller details that don't need to adapt.

Relative Units

%
relative to the parent element. Good all-arounder for for laying out elements and flexible design.
vw
relative to 1% of the width of the viewport. Commonly used for sizing top-level elements (i.e. body, main), or elements that break out of the standard page flow (i.e. modals, overlays).
vh
relative to 1% of the height of the viewport. Similar to vw, but for vertical sizing.
vmin
relative to 1% of the viewport's smaller dimension
vmax
relative to 1% of the viewport's larger dimension

Typographic Units

rem
relative to the font-size of the root (browser) font-size - usually 16px. Best practice for font-sizes other than ones that are intentionally relative to another font-size (in which case, use em)
em
relative to the font-size of the element. Good for reusable elements that may have different parent font sizes, as well as for layout spacing that should be relative to text.
ch
relative to the size of one character in the element's font, specifically the width of the "0" character. Good for sizing elements that should be relative to the text inside them, such as input fields.
