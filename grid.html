"Holy Grail" layout - Two Ways

Header
Left Sidebar
Main Content
Right Sidebar
Footer
CSS - Explicit Grid Lines 

.holy-grail-grid-container {
    display: grid;
    grid-template-columns: 1fr 3fr 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 10px;
    .header {
        grid-column: 1 / -1;
    }
    .left-sidebar {
        grid-column: 1 / 2;
    }
    .main-content {
        grid-column: 2 / 3;
        min-height: 300px;
    }
    .right-sidebar {
        grid-column: 3 / -1;
    }
    .footer {
        grid-column: 1 / -1;
    }
}
CSS - Using Grid Areas 

.holy-grail-grid-container-areas {
  display: grid;
  grid-template-areas:
    "header header header"
    "left-sidebar main-content right-sidebar"
    "footer footer footer";
  grid-template-columns: 1fr 3fr 1fr;
  gap: 10px;
  .header {
    grid-area: header;
  }
  .left-sidebar {
    grid-area: left-sidebar;
  }
  .main-content {
    grid-area: main-content;
    min-height: 300px;
  }
  .right-sidebar {
    grid-area: right-sidebar;
  }

  .footer {
    grid-area: footer;
  }
}
HTML 

<div class="holy-grail-grid-container">
    <div class="grid-item header">Header</div>
    <div class="grid-item left-sidebar">Left Sidebar</div>
    <div class="grid-item main-content">Main Content</div>
    <div class="grid-item right-sidebar">Right Sidebar</div>
    <div class="grid-item footer">Footer</div>
</div>
